## Sunku Giridhar Shanmukh – Futuristic DevOps Portfolio

This is a cyberpunk / gamer-style portfolio for **Sunku Giridhar Shanmukh**, an Associate DevOps Engineer and Full Stack Developer from India. The UI is designed to feel like a high-tech game dashboard with neon highlights, animated gradients, HUD elements, and scroll-triggered animations.

### Tech Stack

- **Framework**: Next.js 14 (App Router) + React 18
- **Styling**: Tailwind CSS with custom cyberpunk theme
- **Animation**: Framer Motion + GSAP

### Getting Started

```bash
npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.

You can customize content in the `components` directory (hero, about, skills, projects, DevOps timeline, GitHub activity, contact, HUD, cursor, and loading screen).
